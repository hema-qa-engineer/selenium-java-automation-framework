# Selenium Automation - Banking Application

This project is a Selenium automation framework built using Java for a demo banking application (ParaBank).

## Tech Stack
- Selenium WebDriver
- Java
- TestNG
- Maven

## Framework
- Page Object Model (POM)
- TestNG for execution

## Scenarios Covered
- Login (valid & invalid)
- Account overview
- Fund transfer
- Bill payment
- User registration

## What I did
- Automated key banking flows like login, transfer, and bill payment
- Used POM to keep code reusable and maintainable
- Handled waits, alerts, and dynamic elements
- Organized test execution using TestNG

## Run Command
mvn clean test
